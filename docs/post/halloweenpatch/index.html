Halloween Patch Update!

31 Oct 2017

I recently pushed out some updates to fix a few issues and add another. Explanations are worked into the documentation page, but changes included:

  • Increasing the radius of the ping functionality. It will now get information on everything in a 8 square radius.
  • Increased the default map size to 20x20 and the default max number of obstacles to 10. These are of course still configurable, but these will be the defaults that the leaderboards use.
  • Trying to squash a bug regarding the self-destruct mode and bots not correctly timing out. Hopefully this is working better now and introduced no new issues.
  • When bots are killed they will now drop all dodgeballs they had on them in the local area. Let the rush to grab everything commence!

Changes have been pushed to Github. If you are using git to stay updated, you just need to do a pull, if you are downloading zip files, you’ll need to grab the latest here and extract and overwrite the old files.
