Halloween Patch Update!

31 Oct 2017

I recently pushed out some updates to fix a few issues and add another. Explanations are worked into the documentation page, but changes included: Increasing the radius of the ping functionality. It will now get information on everything in a 8 square radius. Increased the default map size to 20x20 and the default max number of obstacles to 10. These are of course still configurable, but these will be the defaults that the leaderboards use.
